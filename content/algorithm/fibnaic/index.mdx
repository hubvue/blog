### 题目
**斐波那契数**，通常用 `F(n)` 表示，形成的序列称为 **斐波那契数列** 。该数列由 `0` 和 `1` 开始，后面的每一项数字都是前面两项数字的和。也就是：
```txt
F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
```
给你 `n` ，请计算 `F(n)` 。

**示例 1：**
```txt
输入：2
输出：1
解释：F(2) = F(1) + F(0) = 1 + 0 = 1
```

**示例 2：**
```txt
输入：3
输出：2
解释：F(3) = F(2) + F(1) = 1 + 1 = 2
```

**示例 3：**
```txt
输入：4
输出：3
解释：F(4) = F(3) + F(2) = 2 + 1 = 3
```

提示：
- `0 <= n <= 30`

### 解析
这道题已知题目中已经给出了计算公式，即`F(n) = F(n - 1) + F(n - 2) n >= 2`。基本上所有人一眼就能看出这个一个函数递归调用的问题，但同时这又是一个递推问题，符合动态规划的基本特征：
- 重叠子问题：以计算`F(7)`为例，计算公式为`F(7) = F(6) + F(5)`，而`F(6) = F(5) + F(4)`，因此存在大量的重复计算
- 最优子结构：公式本身就是一个最优结构，当前值的需要通过前面的值计算得到
- 无后效性：从公式本身也可以看出，后面的值最前面的值没有影响

因此这道题既可以用递归来解，又可以用动态规划来解，由于递归太多会触发函数执行栈的溢出，因此主要还是用动态规划来解。

### 代码
```go
func fib(n int) int {
    if n < 2 {
        return n
    }
    a, b := 0, 1
    for i := 2; i <= n; i ++ {
        a, b = b, a + b
    }
    return b
}
```
