Rust 变量遮蔽允许在同一个作用域下重复定义相同的变量，底层逻辑应该是值`生命周期`和变量`行为的改变`。

**生命周期**

```rust
let x = 123;

let x = "Hello world"
```
值`123`生命周期结束，x指向 `Hello world`。

**行为改变**

```rust
let x = 123;

let mut x = x;
```
从只读行为变成可读可写行为。

因此基于变量遮蔽可以改变变量的行为，也可以做一些事情。比如：定义变量可写范围。

```rust
fn main() {
  let mut v = Vec::new(); // 可变的Vec变量
  v.push(1);
  v.push(2);
  v.push(3);
  // 可写范围结束
  let v = v;  // 从现在开始 v变量 只读
  for i in &v {
    println!("{}", i);
  }
}
```
同样的道理，将变量从只读变成可变也是可以的。
