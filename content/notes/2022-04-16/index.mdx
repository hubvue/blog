Rust中使用`let`关键字进行变量的声明，并且是只读的。
```rust
let x: i32 = 123;

// x = 234; // error
```
如果需要对变量进行修改，需要将变量声明成可变的，使用`mut`关键字表示。
```rust
let mut x: i32 = 123;

x = 234;
```

当使用`let`声明只读变量时，编辑器会帮我们做一个执行路径的静态分析，确保变量在使用前一定初始化。因此变量的第一次赋值是初始化而不是修改。
```rust
let x: i32;   // 声明
{
  x = 234;    // 初始化
  println!("x: {}", x);
}
```
这种情况下是不需要`mut`关键字修饰。

这种方式也有一些应用场景。比如：当在做只读变量的初始化是，而变量的初始化操作参杂分支和复杂逻辑，就可以不使用`mut`修饰，只需要保证代码执行路径上没有修改即可。

```rust
fn test(condition: bool) {
  let x: i32;
  if condition {
    // xxxx 一些复杂逻辑
    x = 1;
  } else {
    // xxxx 一些复杂逻辑
    x = 2;
  }

  println!("x: {}", x);
}
```
