### TupleToUnion

在 `ValuesType` 中已经提到一个特别简单的方式。还有一种方式也值得学习一下。

在类型系统中，元组类型是兼容数组类型的。

```ts
// 'true'
type ret = [number, string] extends Array<any> ? 'true' : 'false'
```

因此就可以使用 infer 来推断出数组的泛型类型。

**实现**

```ts
export type TupleToUnion<T extends any[]> = T extends Array<infer U> ? U : never
```
